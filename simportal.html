Collatz

A Scale‑Dynamic View of a Classical Map

AXLE approaches the Collatz map as a behavioral system, not a theorem. Instead of searching for invariants or hidden returns, we study how trajectories move across scales under the simple local rule:

T(n) = n/2 (even),   T(n) = 3n + 1 (odd)

This map exhibits rich structure when viewed through AXLE’s scale lens:

  • Cycle‑scale (g⁶) — short oscillations, parity bursts, local breathing patterns.
  • Equilibrium region (g³³) — a soft drift toward small numbers, with visible variance.
  • Phase‑change thresholds (g⁶⁴) — magnitudes where behavior shifts qualitatively.

AXLE makes no claims about universal convergence. There are no invariants asserted, no returns predicted, and no theorems implied. Collatz is treated as it is: a simple rule generating complex, observable behavior across multiple scales.

Transfer Operator

2-Adic Transfer Matrices for the Collatz Map

Seed Sentence (C1): The Collatz map shuffles residue classes the same way a doubly stochastic matrix shuffles probability — every class sends weight to exactly two targets, every class receives weight from exactly two sources, and the long-run drift is always downward.

The finite-dimensional transfer operator PM encodes where residue classes go under the accelerated 2-adic step T(n) = n/2 (even), T(n) = (3n+1)/2 (odd). For each modulus M, the M×M matrix PM has entry PM[j, i] = 1/2 whenever some n ≡ i (mod M) satisfies T(n) ≡ j (mod M). All entries are either 0 or 1/2; every column sums to 1.

Spectral Radius and Density Scaling Law

  • Doubly stochastic: PM has exactly 2 nonzeros per column and per row, each equal to ½. Both row and column sums equal 1.
  • Spectral radius: ρ(PM) = 1 for all M = 2k, k ≥ 4. The leading eigenvalue is 1 (uniform stationary distribution).
  • Density scaling: nonzero density = 2/M. Doubling M halves the density: 12.5% → 6.25% → 3.125% → …
  • Non-zero count: always 2M entries out of M² — the matrix grows sparser as resolution increases.

Weighted Drift and Bridge 0

The transfer operator perspective connects directly to the log-scale drift. Under the accelerated step T, an integer n carries the weight:

E[log T(n)/n] = ½·log(½) + ½·log(3/2) = ½·log(3/4) ≈ −0.1438

Each step multiplies the 2-adic magnitude by an expected factor of (3/4)½ ≈ 0.866. The drift is strictly negative: on average, every application of T brings n closer to 1 in the 2-adic metric. This is the quantitative content of Bridge 0: the coefficient c = 3 in the rule 3n+1 is precisely the triad fingerprint that keeps the drift below zero — any c ≥ 4 would flip the sign and break convergence.

The polar vortex certificate (triad c = 3, dm³ scaling g⁶⁶) is the geometric counterpart of this arithmetic fact: the same crystal geometry that stabilises Saturn's hexagonal vortex governs the long-run contraction of Collatz trajectories.
