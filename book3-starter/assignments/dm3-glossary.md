# dm³ Glossary — Key Terms

Read this before you begin any assignment. These words appear throughout the book and the assignments.

---

## Core Operators

| Term | Plain-language meaning |
|---|---|
| **dm³** | "Differential Morphogenetic Matrix" — a mathematical framework for describing how patterns grow and transform across scales |
| **C (Compression)** | Reducing information while keeping its essential structure |
| **K (Kernel)** | The core rule or generator that drives a pattern |
| **F (Flow)** | How a pattern spreads or propagates through a space |
| **U (Unification)** | Combining multiple patterns or scales into one coherent structure |
| **Operator chain** | A sequence of operations: C → K → F → U means compress, find the kernel, let it flow, then unify |

---

## Scaling and Hierarchies

| Term | Plain-language meaning |
|---|---|
| **Scaling hierarchy** | A ladder of structure: small patterns that repeat at larger and larger sizes |
| **g⁵ / g⁶** | Notation for levels in the hierarchy (think: "gear 5", "gear 6") |
| **Hyper-Mahlo regeneration** | A mathematical concept: a system that can rebuild itself from its own rules at very high levels of abstraction |
| **Regeneration loop** | A feedback cycle where output feeds back to become input |

---

## Topographical Orthogenetics (TO)

| Term | Plain-language meaning |
|---|---|
| **Topographical** | Related to the shape and layout of a space or structure |
| **Orthogenetics** | Development that follows strict directional rules (from Greek: *ortho* = straight, *genesis* = origin) |
| **TOGT** | Topographical Orthogonal Generative Theory — the full theoretical framework |
| **AXLE** | The formal verification and extension hub for TOGT proofs |

## Bridges (Cross-Domain Applications)

| Domain | What it means in this course |
|---|---|
| **Plasma instabilities** | Chaotic behavior in ionized gases — used as a physical test case for dm³ operators |
| **Biological morphogenesis** | How bodies grow and form — another test case |
| **Martian colony architecture** | Future application: using dm³ to plan self-organizing structures on Mars |
| **Fly connectome** | The *Drosophila* (fruit fly 🪰) brain connection map — smallest known full brain, used as a toy model |

## Technical Terms

| Term | Plain-language meaning |
|---|---|
| **Lean 4** | A programming language for writing and verifying mathematical proofs |
| **Collatz conjecture** | A famous unsolved math problem: pick any positive integer, apply a simple rule repeatedly — does it always reach 1? |
| **Lyapunov stability** | A way to measure whether a system stays "under control" or spirals out |
