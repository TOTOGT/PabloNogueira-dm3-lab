## 🚀 How to Get Started (Step by Step)

1. **Click the assignment link** your teacher shared.
2. **Sign in to GitHub** — Classroom creates your repo automatically.
3. Open in **GitHub Codespaces** (green `<> Code` button → Codespaces tab → *New codespace*),
   OR clone to your computer:
   ```bash
   git clone https://github.com/TOTOGT/<your-repo-name>.git
   cd <your-repo-name>
   ```
4. Read `assignments/dm3-glossary.md` first — it explains the key ideas.
5. Open the folder for your current week (`assignments/A1/`, etc.).
6. Read `README.md` inside that folder for the prompt and instructions.
7. Write your answers / code in that folder.
8. **Save and submit:** commit your changes and push:
   ```bash
   git add .
   git commit -m "A1 – my first submission"
   git push
   ```

> **Pair / Group work:** Only one team member needs to push. Everyone else will see the update automatically.
